# TMA — מחשבון שיפוצים (Rehab Cost Calculator)

A self-contained, RTL Hebrew rehab-cost estimator for The Multi Academy students.
Single `index.html`, no build step. Property intake → tier-based Scope of Work →
results dashboard with live KPIs → styled Excel export. Works offline after load
(saves progress in your browser); one external CDN (ExcelJS) is used only for the
Excel download.

**Live app:** https://ohadozfam-design.github.io/tma-rehab-calculator/
